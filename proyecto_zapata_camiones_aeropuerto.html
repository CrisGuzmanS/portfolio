Zapata Camiones Aeropuerto
DESCRIPCIÓN

Participé en el desarrollo de un sistema e-commerce de venta de autopartes para Zapata Camiones Aeropuerto. Además implementé en la implementación de un sistema administrativo para clientes, productos, y ventas.

Caracteristicas principales
  • Pagos mediante Paypal
  • Pagos mediante NetPay
  • Importacion / Exportacion Excel
  • Envío de correo
  • Gráficas de reportes (reportes)
  • Perfiles de acceso
