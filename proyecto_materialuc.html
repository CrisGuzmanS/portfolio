Participé en la implementación un sistema web que se encarga de controlar los préstamos de dispositivos y herramientas de la División de Ingenierías Civíl y Geomática, mediante escaneo de QR para usuarios y materiales. Los lenguajes y frameworks utilizados fueron PHP, MySQL, javascript, CSS, HTML y bootstrap.

Las tareas que realicé en este proyecto fueron el desarrollo del frontend del sistema web completo, el desarrollo de las funciones CRUD para la administración de usuarios y materiales, la implementación del sistema de login mediante escaneo de QR, así como la lógica de programación para el control de préstamos de los materiales.
